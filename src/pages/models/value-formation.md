# Value Formation
### Why some things matter more than others.

## The Core Idea

What feels important is not random.

It is not just what is loud, recent, or everywhere.

Things matter when they actually change something.

The more something shifts behavior, systems, or outcomes, the more weight it carries.

## The Pattern

You can think of value as coming from three things:

Value = Impact x Reach x Persistence

This is not a strict formula.

It is a way to see why some things matter more than others.

## Impact

What does it actually change?

Impact shows up when something affects:

- money
- safety
- resources
- daily behavior
- decisions

If nothing changes, the impact is low, no matter how much attention it gets.

## Reach

How far does it spread?

A small event affects a few people.

A larger event affects many.

The wider it spreads, the more systems it touches.

## Persistence

How long does it last?

Some things fade quickly.

Others keep shaping behavior over time.

The longer it stays active, the more weight it builds.

## Putting It Together

Something becomes meaningful when it:

- changes something real  
- affects more than one place  
- continues over time  

That is what gives it weight.

## Why This Matters

Attention is not the same as importance.

Something can be everywhere and still change nothing.

Something else can be quiet and still reshape everything.

This helps you see the difference.

## Use This

When something feels important, ask:

- what is this actually changing?
- how far does it reach?
- how long will this last?

That will tell you more than how loud it feels.

## Final Line

What matters is not what gets attention.

It is what actually changes things.
